# 🤖 AI Customer Support Chatbot with Lead Capture

## 🚀 Project Overview
This is a smart AI chatbot built using Flask that can:
- Answer user queries
- Capture user details (Name & Email)
- Store leads in a database

## 💡 Features
- Chat interface (HTML, CSS, JS)
- Flask backend
- Lead capture system
- SQLite database
- Fallback responses (no API needed)

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- SQLite
