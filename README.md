# 🤖 AI Customer Support Chatbot with Lead Capture

## 🚀 Project Overview
This is a smart AI chatbot built using Flask that can:
- Answer user queries
- Capture user details (Name & Email)
- Store leads in a database

## 🌟 Live Features
- 💬 Smart chatbot interface
- 📩 Lead capture system (Name & Email)
- 🧠 AI-ready architecture
- ⚡ Fast Flask backend

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- SQLite
